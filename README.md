# Patient-Level Predictive (PLP) Modeling with the Veterans Precision Oncology Data Commons (VPODDC)

This repository contains cohort definitions and PLP model designs described in the manuscript "Patient-Level Predictive Modeling with the Veterans Precision Oncology Data Commons."


## Cohort definitions

Please see cohort definitions used in the manuscript in the [cohorts](./cohorts/) folder:
* dataset.json - initial study population
* outcome.json - population with outcome of interest

## Model design metadata

Please see model designs used in the manuscript in the [model_designs](./model_designs/) folder.  There is metadata for Lasso Logistic Regression, Random Forest, and Support Vector Machine models for 3, 5, and 10 years.
